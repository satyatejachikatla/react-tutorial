# React Sample Projects

## Commands List

### Create React App

```
npx create-react-app <app-name>
```

### Start https://localhost:3000

```
cd <app-name>
npm start
```

### For auto reaload of react

```
cd <app-name>
echo "FAST_REFRESH=false" > .env
```
